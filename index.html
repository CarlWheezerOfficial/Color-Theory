Color Theory

What is Color Theory?

Color Theory is a set of rules that designers use to decide color choices, color can effect how users feel, experience, and interact with a webpage.

Color can lead the users eye, or make them feel a certain way.

I know the warm and dark colors in this website create a cozy atmospere.

And I know this box is likely the first place you looked.

Good color choice is essential to design, a good balance between contrast and harmony can create a readable and asthetic webpage.

Color Theory is used in almost everything, from artwork, to branding, to interior design.

A good example is restraunts that use the color red in their branding, because there is coorelation between the color red and food.
